About this project

Swiss Will is a small, framework-free static website that explains the options available for inheritance and estate planning in Switzerland.

Purpose

The goal is to present Swiss succession concepts in clear, approachable language so that people can understand the broad shape of their choices before speaking to a qualified professional. The current site is an early placeholder shell — the wording on each page is illustrative and will be rewritten.

Scope

  • Intestate succession (when there is no will)
  • Wills — holographic and public deed
  • Inheritance contracts (Erbvertrag)
  • Forced heirship and the disposable portion

How it is built

The site is hand-written HTML, CSS, and a small amount of vanilla JavaScript — no framework and no build step. It is hosted on Cloudflare Pages as static files. See the repository and its README.md for details.

Disclaimer: Nothing on this site is legal advice. Swiss succession law is governed by the Civil Code (ZGB/CC) and can change. Always consult a qualified notary or lawyer for your own situation.
